# Sorting Algorithm Visualizer

A professional, interactive **Sorting Algorithm Visualizer Web Application** built using **React**.  
This project helps users understand sorting algorithms through **step-by-step visual animations**, **theoretical explanations**, **complexity analysis**, and **algorithm comparisons**, with support for **dark mode** and **real-time controls**.

---

## 🚀 Features

- 📊 Interactive visualization of sorting algorithms
- ⏯️ Start / Stop controls for animations
- ⚡ Adjustable animation speed
- 🌙 Light / Dark mode (applies to the entire application)
- 📘 Dedicated theory pages for each algorithm
- 📈 Side-by-side algorithm comparison table
- 🧭 Multi-page dashboard with routing
- 🎨 Clean, professional UI with hover effects
- 📱 Responsive layout

---

## 🔢 Sorting Algorithms Implemented

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

Each algorithm includes:
- Step-by-step visualization
- Best / Average / Worst time complexity
- Space complexity
- Stability and in-place information

---

## 🧠 Learning Sections

- **Home Dashboard**
  - About sorting
  - Algorithm cards
  - Navigation to visualizer and comparisons

- **Algorithm Details Pages**
  - Explanation of how the algorithm works
  - Algorithm steps
  - Time & space complexity analysis

- **Comparison Page**
  - Side-by-side comparison of all algorithms

## 🛠️ Tech Stack

- **Frontend:** React (Functional Components & Hooks)
- **Routing:** React Router DOM
- **Styling:** CSS with theme variables
- **State Management:** React `useState`, `useRef`
- **Icons:** Devicon & CDN icons
- **Theme Handling:** CSS Variables + Local Storage
